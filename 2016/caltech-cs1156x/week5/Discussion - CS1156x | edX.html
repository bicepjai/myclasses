        3. Sounds right. I think you're talking about the second to last slide from week 5 -- slide 23. It sums up N terms into a combined gradient vector and then multiplies that by a scalar of 1/ N -- which is what we do in gradient descent when we are using a batch with N terms in it.

          In stochastic gradient descent, in effect we are only looking at a randomly selected 'sub-batch' with 1 data point in it. So here we are adding up 1 terms and then multiplying that by a scalar of 1/1 ... and since muliplying by 1 has no impact we can just skip it.

                HW5 PROBLEM 1

                If we call the expected in-sample error EEin we have:

                N=d+11EEin2

                and using the problem data this gives N about 45.
                EEin increases with N, thus the smallest value above 45 that gives an expected error above 0.008, is 100.

                HW5 PROBLEM 2

                Using the given transformations we can get hyperbolae with the center in the origin and axis coincident with the coordinate system. They have this equations:

                x2a2y2b2=1b2x2a2y2a2b2=0

                The equation maps the points in the opposite way we want (i.e. (0,0) is negative) so we must change its sign as:

                b2x2+a2y2+a2b2=0w0+w1x2+w2y2=0

                with:

                w0=a2b2>0w1=b2<0w2=a2>0

                HW5 PROBLEM 10

                According to the PLA, choosing =1 for SGD, we need an error function E with a gradient E such that:

                1. wwE=w when a point is correctly classified;
                2. wwE=w+ynxn when a point is misclassified.

                In this case our gradient must be:

                E={0if point is correctly classifiedynxnif point is incorrectly classified

                If we consider e) en(w)=min(0,ynwTxn) we have such behaviour. In fact if we explicitate the min function error is:

                en(w)={0ynwTxn>0ynwTxnynwTxn<0

                where ynwTxn>0 is exactly the condition for a correctly classifed point. Now if we calculate the gradient (in w=0 the function is not differentiable but we can ignore):

                en(w)={0ynwTxn>0ynxnynwTxn<0

                as we needed.

                  R code for Q8 and Q9 :

                  ### Define some functions ###
## Cross entropy error ##
 E <- function(x,y,z) {log(1+exp(-y%*%z%*%x))}
## Gradient ##
grad <- function(x,y,z){(-y%*%x)/c(exp(y%*%z%*%x)+1)}
## Weights vector variation##
norm_vec <- function(x) sqrt(sum(x^2))

### The algorithm ###
run <- 1
epoch.vec <- NULL
Eout.vec <- NULL
while(run<100) # Run the algorithm for 100 times
{
 ## Find a boundary line ##
 x1.l <- runif(2,-1,1)
 x2.l <- runif(2,-1,1)
 slope.l <- diff(x2.l )/diff(x1.l)
 intercept.l <- x2.l[1]-slope.l*x1.l[1]
 ## Create 100 random points and evaluate y ##
 x1.p <- runif(100,-1,1)
 x2.p <- runif(100,-1,1)
 intercept.p <- rep(1,100)
 data <- cbind(intercept.p,x1.p,x2.p)
 y.p <- ifelse(x2.p>intercept.l+slope.l*x1.p,+1,-1)  
 ## Initialize weights and epoch ##
 ## Create a vector to collect the final weights after each epoch ##
 w <- c(0,0,0) #Initialize weights to 0
 w.vec <- NULL
 w.vec <- rbind(w.vec,w)
 rate <- 0.01
 epoch <- 1
 w.var <-1 # Initialize to some values greater than the threshold
 thres <- 0.01 
 while (w.var>=thres) #convergence criteria
 {
   seq <-sample(c(1:100),100,replace=F) # Random permutation of 1-100
 ## Stochastic gradient descent  ##  
   for (i in 1:100)  
   {
     gradient <- grad(data[seq[i],],y.p[seq[i]],w)
     w <- w -rate%*%gradient
   }
   w.vec <- rbind(w.vec,w)
   epoch = epoch+1
   w.var <-norm_vec(w.vec[epoch-1,]-w.vec[epoch,])
 }
 ## Eout ##
 ## Create 1000 random points and evaluate y ##
 x1.out <- runif(1000,-1,1)
 x2.out <- runif(1000,-1,1)
 intercept.out <- rep(1,1000)
 data.out <- cbind(intercept.out,x1.out,x2.out)
 y.out <- ifelse(x2.out>intercept.l+slope.l*x1.out,+1,-1)
 Eout <- NULL
 ## Calculate the average cross entropy error for the 1000 points ##
 for (i in 1:1000)
 {Eout[i] <- E(data.out[i,],y.out[i],w.vec[epoch,])}
 Eout.vec<- cbind(mean(Eout),Eout.vec)
 epoch.vec <- cbind(epoch,epoch.vec)
 run = run+1

                  }

                  mean(Eout.vec) # Ans to Q8 
mean(epoch.vec) # Ans to Q9
